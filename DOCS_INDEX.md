## API Endpoints Quick Reference

### Post Procedures

| Procedure | Usage | Returns |
|-----------|-------|---------|
| `post.getAll` | Query posts with pagination/search/filters | `{ posts: [], total }` |
| `post.getBySlug` | Get single post by slug | Post or null |
| `post.getById` | Get single post by ID | Post or null |
| `post.getByCategoryId` | Get posts in category | Post[] |
| `post.create` | Create new post | Post |
| `post.update` | Update post | Post |
| `post.delete` | Delete post | `{ success, deletedId }` |

### Category Procedures

| Procedure | Usage | Returns |
|-----------|-------|---------|
| `category.getAll` | Get all categories | Category[] |
| `category.getBySlug` | Get category by slug | Category or null |
| `category.getById` | Get category by ID | Category or null |
| `category.create` | Create category | Category |
| `category.update` | Update category | Category |
| `category.delete` | Delete category | `{ success }` |
| `category.getPostsByCategory` | Get posts in category | Post[] |
